# 4X Strategy Game - Wolf Game Style

Mobil 4X stratejik oyun: Köy geliştirme, klan sistemi, harita kontrol ve gerçek zamanlı oyun mekanikleri.

## 🎮 Oyun Özellikleri

- **Köy Geliştirme**: Binalar inşa etme, kaynak yönetimi
- **Klan Sistemi**: Oyuncular klan oluşturabilir ve büyük haritada birlikte yer kapabilir
- **Harita Kontrolü**: Stratejik konum seçimi ve savunma
- **Gerçek Zamanlı**: Real-time oyun mekanikleri
- **Video Ödülleri**: Kullanıcı isteğiyle video izleme (Admob entegrasyonu)
- **Cross-Platform**: iOS ve Android desteği

## 🛠️ Teknoloji Stack

- **Engine**: Unity 2022 LTS+
- **Grafik**: 2D (ağırlık) + 3D efektleri
- **Ağ**: Photon PUN 2 / Mirror (Multiplayer)
- **Veri**: Firebase Realtime Database
- **Reklamlar**: Google Admob

## 📁 Proje Yapısı

```
Assets/
├── Scripts/
│   ├── Core/           # Oyun çekirdeği
│   ├── Managers/       # Game, Network, Data yöneticileri
│   ├── UI/             # UI sistemi
│   ├── Buildings/      # Bina sistemleri
│   ├── Map/            # Harita sistemi
│   ├── Clan/           # Klan sistemi
│   ├── Monetization/   # Ads, Video rewards
│   └── Utilities/      # Yardımcı fonksiyonlar
├── Prefabs/
├── Sprites/
├── Materials/
├── Scenes/
├── Resources/
└── Plugins/
```

## 📦 Kurulum

1. Unity 2022 LTS veya üzeri ile proje açın
2. TextMesh Pro'yu import edin
3. Package Manager'dan gerekli paketleri yükleyin
4. Firebase ve Admob kurulumunu tamamlayın
